# Energy-Efficient CPU Scheduling Algorithm

## Project Overview
This project focuses on the designing a CPU scheduling algorithm that reduces energy consumption while maintaining performance. The algorithm simulates different CPU frequency levels and selects the most power-saving option based on workload.

## Features
- Simulated CPU scheduling
- Dynamic Voltage and Frequency Scaling (DVFS)
- Task energy calculation
- Visualization-ready data
